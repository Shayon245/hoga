This will open the admin panel in a new window for manual testing.

Manual Testing Steps:

  1. Click "Open Admin Panel" above
  2. In the admin panel, click "Schedules" in the sidebar
  3. Click "Add Schedule" button
  4. Fill in all form fields:
    • Bus: Select any option
    • Route: Select any option
    • Departure Time: e.g., 10:00
    • Departure Date: Select today or future date
    • Fare: e.g., 500
    • Available Seats: e.g., 40
    • Status: Active
  5. Click "Add Schedule"
  6. You should see "Schedule added successfully!" message
  7. The new schedule should appear in the table
